## 🚀 주요 기능
- 25분 집중 + 5분 휴식 기본 타이머
- 타이머 진행 상황 시각적으로 표시
- 타이머 종료 시 진동 및 알림으로 다음 세션 안내
- 세로/가로 모드 지원
- 집중 시간, 짧은 쉬는시간, 긴 쉬는시간 설정
- 취향에 따른 테마 설정
- 세션 간 자동 전환 설정 및 진동 모드 전환 설정
