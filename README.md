## 🚀 주요 기능
- 25분 집중 + 5분 휴식 기본 타이머
- 타이머 진행 상황 시각적으로 표시
- 집중 모드를 방해하지 않는 간결한 인터페이스
- 세로/가로 모드 지원
- 세션 간 자동 전환으로 손쉬운 흐름 유지
- 타이머 종료 시 진동 및 알림으로 다음 세션 안내
